This app contains a player field (quota_full, initially set to 0), a participant field (quota_full) and a session field (num_participants_finished).

When creating a session, num_participants_finished is set to 0 to start the counter.

After the first page (e.g. informed consent) quota_full is stored as a participant variable. This is only neccessary when quota_full needs to be retrieved in a different app. However, a page is needed prior to the following page.
